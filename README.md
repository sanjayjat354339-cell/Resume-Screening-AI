# 🚀 Resume Screening AI

**Live deployed Resume Screening AI that classifies resumes into job roles using NLP and Machine Learning.**

🚀 **Live Demo:**  
👉 https://resume-screening-ai-a2fn.onrender.com/

---

## 📌 About Project

A Machine Learning + NLP based web application that analyzes resume text and predicts the most suitable **job role** along with a **confidence score**.

---

## 🧠 Features

- 📄 Input resume text  
- 🤖 Predicts job role  
- 📊 Shows confidence level (%)  
- ⚡ Deployed on Render  
- 🧠 NLP-based processing  

---

## 🛠️ Tech Stack

- Python  
- Flask  
- Scikit-learn  
- NLTK  
- TF-IDF Vectorizer  

---

## 📂 Project Structure
Resume-Screening-AI/ │── templates/ │── app.py │── model.py │── preprocess.py │── data.py │── model.pkl │── encoder.pkl │── tfidf.pkl │── requirements.txt │── README.md
---

## ⚙️ How It Works

1. User enters resume text  
2. Text is cleaned using NLP  
3. TF-IDF converts text into features  
4. Model predicts job role  
5. Confidence score is displayed  

---

## 📊 Output

- 🎯 Predicted Job Role  
- 📈 Confidence Score (%)
