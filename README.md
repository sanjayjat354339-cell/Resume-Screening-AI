📌 Resume Screening AI
A Machine Learning + NLP based web application that analyzes resume text and predicts the most suitable job role along with a confidence score.
🚀 Live Demo:
👉 https://your-render-link.onrender.com⁠�
🧠 Features
📄 Input resume text
🤖 Predicts job role based on resume content
📊 Displays confidence level (%)
⚡ Deployed web application (Render)
🧠 Uses NLP techniques for text processing
🛠️ Tech Stack
Python
Flask
Scikit-learn
NLTK
TF-IDF Vectorizer
Render (Deployment)

⚙️ How It Works
User enters resume text
Text is cleaned using NLP preprocessing
TF-IDF converts text into numerical features
Trained ML model predicts job role
Confidence score is calculated and shown
📊 Output
🎯 Predicted Job Role (e.g., Data Scientist, Web Developer)
📈 Confidence Score (%)
